# PirateWorld — CANON v1 (Estable)

Esta carpeta contiene la definición CANON v1 del sistema PirateWorld.
Todo desarrollo posterior debe respetar estrictamente estos documentos
o abrir un sprint explícito de cambio CANON.

CANON v1 congela:
- FSM principal
- Contrato del CTA
- Contrato del HUD
- PROFILE (read-only con datos reales)
- EVENT como overlay BLOCKING
- Reglas operativas FSM-first / observer-only
